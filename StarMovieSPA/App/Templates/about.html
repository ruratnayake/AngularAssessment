Assessment - #DEV01: Exilesoft Angular developer certificate

This is a demo application developed using AngularJS 1 along with html and css as a part of the Exilesoft Angular developer certificate program, in order to get certified as Exilesoft Angular developer.

Assignment

Create an app that lists your favorite movies. The app should contain a list of the movies, details of each movie as well as the top/favorite characters. The following list highlights the main requirements:

  • The movie list should show the name and your preferred rating for each movie
  • The movie details should contain: Title, Director, Release Date, Rating, Language and Country
  • Character list should contain: Character Name and the Actor/Actress Name
  • A form to add movie details including the characters

Try to use as many features as possible in AngularJS (controller, service, routing, inbuilt/custom directives, etc.).
